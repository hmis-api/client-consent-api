/clients/{clientid}/consentrequests/{consentrequestid}/statuses put

put

Approve or Reject client consent request. Once consent request is approved a new consent will be created for the requester project group and for the consent entity. The consent request, cannot be modified any further as it is already approved. Any changes to the duration of the consent etc. can be done on the consent directly. Allowed values for the consent request status are APPROVED / REJECTED.
